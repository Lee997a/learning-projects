API 테스트

브라우저 개발자 도구(F12)를 열고 다음 JavaScript 코드를 실행해보세요:

// 사용자 전용 API 테스트
fetch('/api/user/test', {
    method: 'GET',
    headers: {
        'Content-Type': 'application/json'
    }
})
.then(response => response.text())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
JWT 토큰이 쿠키에 자동으로 포함되어 인증됩니다.
